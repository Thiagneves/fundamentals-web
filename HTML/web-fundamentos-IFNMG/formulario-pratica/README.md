## 1. Cadastro de Usuário

Crie uma página HTML com um formulário de cadastro de usuário contendo:

* nome completo
* e-mail
* senha
* botão de enviar

### Regras:

* cada campo deve ter seu respectivo `<label>`
* os campos devem ser obrigatórios
* o formulário deve ter um título principal com `<h1>`

## 2. Inscrição em Curso

Crie um formulário para inscrição em um curso de informática. O formulário deve conter:

* nome completo
* idade
* e-mail
* telefone
* curso desejado com `<select>` e pelo menos 4 opções
* botão de enviar
* botão de limpar

## 3. Formulário de Contato

Crie uma página com um formulário de contato para um site institucional. O formulário deve ter:

* nome
* e-mail
* assunto
* mensagem com `<textarea>`

### Regras:

* a mensagem deve permitir um texto maior
* todos os campos devem possuir `<label>`
* adicione um botão para enviar

## 4. Agendamento de Atendimento

Crie um formulário para agendamento de atendimento. Inclua os campos:

* nome do cliente
* data do atendimento
* horário
* telefone
* tipo de atendimento com `<select>`
* observações com `<textarea>`

No final, adicione:

* botão “Agendar”
* botão “Limpar”

## 5. Pesquisa de Turno

Crie um formulário de pesquisa com a pergunta: Qual turno você prefere para estudar?

Opções:

* manhã
* tarde
* noite

### Regras:

* use input type="radio"
* somente uma opção pode ser marcada
* adicione um botão de envio

## 6. Evento de Tecnologia

Crie um formulário para um evento de tecnologia com a pergunta: Quais áreas você tem interesse?

Opções:

* programação
* design
* robótica
* banco de dados
* redes

### Regras:

* use input type="checkbox"
* adicione nome e e-mail antes das opções
* finalize com botão de envio

## 7. Matrícula Escolar

Crie um formulário de matrícula para uma escola fictícia. Inclua:

* nome do aluno
* data de nascimento
* nome do responsável
* telefone do responsável
* série desejada com `<select>`
* turno com radio
* observações com `<textarea>`

### Regras:

* agrupe os campos usando `<fieldset>` e `<legend>`
* use pelo menos 2 fieldset

## 8. Envio de Atividade

Crie um formulário para envio de atividade escolar. O formulário deve conter:

* nome do aluno
* turma
* disciplina
* título do trabalho
* campo para selecionar arquivo com input type="file"
* comentário adicional com `<textarea>`

Ao final, adicione um botão “Enviar trabalho”.

## 9. Participação em Evento

Crie um formulário para cadastro de participação em um evento. Inclua:

* nome do participante
* e-mail
* nome do evento
* data
* horário
* tipo de participação com radio:

  * ouvinte
  * expositor
* áreas de interesse com checkbox
* envio de arquivo com input type="file"

## 10. Consulta Médica

Crie um formulário para marcar uma consulta médica. O formulário deve ter:

* nome do paciente
* data de nascimento
* telefone
* e-mail
* tipo de consulta com `<select>`
* plano de saúde com campo de texto
* sintomas ou observações com `<textarea>`

### Regras:

* use `<fieldset>` e `<legend>`
* coloque botões de enviar e limpar
